# Data Directory

This directory contains data files generated from simulations or used as input for analysis.

## Organization

- **Raw data**: Unprocessed simulation outputs
- **Processed data**: Analyzed and cleaned data
- **Input data**: Configuration files and initial conditions

## File Formats

Common formats used:
- CSV for tabular data
- HDF5/JLD2 for large datasets
- JSON for configuration files

## Notes

- Large data files may be excluded from git (see `.gitignore`)
- Include data descriptions and metadata
- Document data generation parameters
